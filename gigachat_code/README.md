# GigaChat Code Assistant

Аналог Claude Code с использованием GigaChat API. MVC архитектура с удобным терминальным интерфейсом.

## Возможности

- 🤖 Чат с GigaChat для помощи в программировании
- 📁 Анализ файлов с кодом
- 💡 Объяснение и рефакторинг кода
- 🧪 Генерация тестов
- 🎨 Красивый терминальный интерфейс на Rich
- 🔐 Безопасное хранение учетных данных

## Установка зависимостей

```bash
pip install rich requests
```

## Быстрый старт

### 1. Настройка API

Запустите мастер настройки:

```bash
python -m gigachat_code.cli setup
```

Вам понадобятся **Client ID** и **Client Secret** от GigaChat API:

1. Зайдите на [портал разработчика Sber](https://developers.sber.ru/)
2. Создайте проект или выберите существующий
3. Подключите продукт **GigaChat**
4. Скопируйте Client ID и Client Secret
5. Введите их в мастере настройки

### 2. Запуск чата

```bash
python -m gigachat_code.cli chat
```

### 3. Просмотр конфигурации

```bash
python -m gigachat_code.cli config
```

## Команды в чате

### Основные
| Команда | Описание |
|---------|----------|
| `/help` | Показать справку |
| `/setup` | Настроить API ключи |
| `/config` | Показать конфигурацию |
| `/quit`, `/exit` | Выйти из программы |

### Работа с чатом
| Команда | Описание |
|---------|----------|
| `/clear` | Очистить историю чата |
| `/new` | Начать новый чат |
| `/history` | Показать историю сообщений |

### Работа с кодом
| Команда | Описание |
|---------|----------|
| `/analyze <путь>` | Анализировать файл |
| `/explain <код>` | Объяснить код |
| `/refactor <код>` | Предложить рефакторинг |
| `/test <код>` | Создать тесты |

## Примеры использования

```bash
# Анализ файла
/analyze my_script.py

# Объяснение кода
/explain def factorial(n): return 1 if n <= 1 else n * factorial(n-1)

# Рефакторинг
/refactor x = [i*2 for i in range(10) if i > 5]
```

## Архитектура

Проект использует **MVC архитектуру**:

```
gigachat_code/
├── models/           # Модели данных
│   ├── config.py     # Конфигурация
│   └── message.py    # Сообщения, сессии
├── views/            # Представление
│   └── terminal_view.py  # Интерфейс на Rich
├── controllers/      # Контроллеры
│   ├── setup_controller.py  # Настройка API
│   └── chat_controller.py   # Управление чатом
├── core/             # Ядро
│   ├── api_client.py      # Клиент GigaChat API
│   └── code_assistant.py  # Бизнес-логика
└── cli.py            # Точка входа
```

## Структура запроса к API

Приложение отправляет запросы в формате JSON:

```json
{
  "model": "GigaChat",
  "messages": [
    {"role": "system", "content": "Ты опытный разработчик..."},
    {"role": "user", "content": "Объясни этот код..."}
  ],
  "temperature": 0.7,
  "stream": false
}
```

## Хранение конфигурации

Учетные данные сохраняются в файле `~/.gigachat_code/config.json` с правами доступа `600` (только владелец).
